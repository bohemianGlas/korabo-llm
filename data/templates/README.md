# プロンプトテンプレート集

サブLLMロールプレイを活かした小説執筆のための雛形です。

| ファイル | 用途 | 貼り付け先 |
|---|---|---|
| `meta_prompt_guide.md` | **大規模LLMにプリセット一式を生成させる**メタプロンプト | ChatGPT/Claude等へ全文＋要望を貼る → 出力を `<id>.preset.md` で保存しプリセットタブでインポート |
| `master_prompt_novel.md` | 小説執筆向けマスタープロンプト（完成版） | 「🧠 マスター設定」タブ → マスタープロンプト欄 |
| `situation_template.md` | シチュエーション雛形（時代・場所・状況） | 「▶ 実行」タブ → シチュエーションプロンプト欄 |
| `role_template.md` | キャラクタープロンプト雛形 | 「🎭 ロール管理」タブ → ロールプロンプト欄 |
| `examples/situation_example.md` | シチュエーション記入例（大正・横浜・探偵もの） | 参考用 |
| `examples/role_example.md` | キャラクター記入例 | 参考用 |

## 使い方

1. 雛形をコピーし、`（…）` のプレースホルダを埋める。**不要な項目は行ごと削除してよい**（空欄のまま残すよりノイズが減って精度が上がる）
2. シチュエーションは長すぎるより「発端のシーン」が具体的なほうが立ち上がりが良い
3. キャラクターは「口調の例文」を2〜3個入れると、サブLLMの声が安定する
4. 「秘密」欄は本人しか知らない情報。陣営機能・心の声と組み合わせると駆け引きが生まれる
